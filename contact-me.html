Contact me

If you have any trouble with the tutorial or you would like to improve it, there are a few ways to contact me:

  • Create an issue on GitHub.

    This is probably the best choice if you had trouble following the tutorial, and something in it should be explained better. Click here to create an issue. You will be asked to create a GitHub account if you don't already have one.

  • Tell me on IRC.

    I'm usually on ##learnpython and ##python-friendly on freenode. See Getting help for instructions to getting there.

If you have trouble with this tutorial please tell me about it and I'll make this tutorial better. If you like this tutorial, please give it a star.
